# Raspberry Pi ネットワーク設定メモ

`Script/interface.py` の `RASPI_URL` で参照する Raspi の IP を記録する。

| 接続方式 | IP アドレス     | ポート | 備考                     |
| -------- | --------------- | ------ | ------------------------ |
| 無線     | `10.27.72.53`   | 9000   | 現在使用中 (2026-05-02)  |
| 有線     | `192.168.10.2`  | 9000   | 有線接続に戻したとき用   |

## 切り替え方法

`Script/interface.py` の `RASPI_URL` を書き換える:

```python
RASPI_URL = "http://<IP>:9000/command"
```
